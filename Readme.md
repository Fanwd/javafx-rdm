# javafx-rdm
An redis client tools developed with JavaFX

#### 项目介绍
一款redis可视化客户端

#### 项目功能
1. 支持新建/删除连接;
2. 支持连接信息的自动保存、加载；

#### 软件架构

1. 技术组件

javaFX、spring-boot、lettuce(redis client)、fontawesomefx(图标库)、dom4j(存储)

2. 构建工具

maven
